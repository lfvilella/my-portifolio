# my-portifolio

Personal portfolio — React (Vite) static site.

Live: https://lfvilella.com

## Local development

Run `npm install`, then `npm run dev`, and open http://localhost:3000

## Tests

Run `npm test`

## Production build

Run `npm run build`, then `npm run preview`

## GitHub Pages

Deployments run via `.github/workflows/deploy-github-pages.yml`. Repository **Pages** source must be **GitHub Actions**. `public/CNAME` configures the custom domain.
